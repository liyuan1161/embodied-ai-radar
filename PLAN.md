# 具身智能商业机会雷达 Implementation Plan

> **For Hermes:** 直接在 `/Volumes/dev/具身智能` 下实现一个可上线的静态变现项目。

**Goal:** 做一个可快速变现的具身智能行业情报/项目机会订阅落地页，收集潜在客户微信/邮箱，售卖报告、订阅和咨询服务。

**Architecture:** 纯静态站点，便于直接用 GitHub Pages、Vercel、Cloudflare Pages 或任意对象存储部署。前端包含价值主张、样例机会、定价、FAQ、线索表单和本地导出 CSV 能力。

**Tech Stack:** HTML + CSS + Vanilla JavaScript，无后端依赖。

---

## 产品定位

名称：具身智能商业机会雷达

目标用户：
- 想进入具身智能方向的创业者
- 关注机器人/空间智能/AI硬件的投资人
- 企业创新部门、AI产品负责人
- 想做副业/咨询/IP内容的人

可卖产品：
1. 199 元：月度机会报告
2. 999 元：季度项目库 + 需求清单
3. 4999 元：1v1 商业化咨询/选题陪跑

## MVP 页面模块

1. Hero：一句话说明价值
2. 痛点：信息太碎、方向太热但不知道怎么落地
3. 机会卡片：展示 3 个可执行商业方向
4. 报告样例：提供免费样章
5. 定价：199 / 999 / 4999
6. 线索表单：姓名、微信、关注方向、预算
7. FAQ：说明不是荐股、不是承诺收益，而是商业情报和咨询

## 验证方式

- 本地打开 `index.html`
- 填表后能在页面看到提交成功
- 能导出 CSV 线索
- README 说明如何部署和开始收钱
