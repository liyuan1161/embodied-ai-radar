# 具身智能商业机会雷达

一个可以快速上线变现的静态项目：售卖具身智能方向的商业情报报告、项目库和咨询服务。

## 项目文件

- `index.html`：落地页
- `style.css`：页面样式
- `app.js`：机会卡片渲染、线索收集、CSV 导出
- `sample-report.md`：免费样例报告
- `PLAN.md`：项目规划

## 如何部署

### 最简单：本地打开

直接双击 `index.html`。

### 上线到 Vercel

1. 新建 GitHub 仓库
2. 上传本目录文件
3. Vercel 导入仓库
4. Framework 选择 `Other`
5. Deploy

### 上线到任意静态服务器

把这些文件上传即可：

```text
index.html
style.css
app.js
sample-report.md
```
